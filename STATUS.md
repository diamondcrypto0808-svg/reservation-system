## ⚠️ 次に必要な作業

### データベースセットアップ
```bash
# PostgreSQLをインストール後
npx prisma migrate dev --name init
npm run db:seed
```

### 環境変数の設定
`.env`ファイルで以下を設定：
- ✅ DATABASE_URL（PostgreSQL接続URL）
- ⚠️ GOOGLE_CLIENT_ID（Google OAuth）
- ⚠️ GOOGLE_CLIENT_SECRET（Google OAuth）
- ⚠️ STRIPE_SECRET_KEY（Stripe）
- ⚠️ NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY（Stripe）
